# Contributing to Linux Bash Labs

Thank you for your interest in contributing to this project! This repository is designed to help people learn Linux and Bash through daily practice.

## How to Contribute

### 1. Add New Labs or Exercises
- Create well-documented exercises
- Include clear learning objectives
- Provide practical examples
- Test all commands before submitting

### 2. Improve Existing Content
- Fix typos or errors
- Clarify confusing explanations
- Add more examples
- Improve formatting

### 3. Share Your Experience
- Submit your practice notes
- Share tips and tricks
- Document common pitfalls
- Suggest improvements

## Contribution Guidelines

### Lab Exercise Format
```markdown
# Lab Title

## Objectives
- Clear learning goals

## Commands to Practice
```bash
# Commands with explanations
```

## Exercises
1. Step-by-step instructions
2. Expected outcomes
3. Challenge variations

## Tips
- Helpful hints
- Common mistakes to avoid
```

### Pull Request Process
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request with clear description

### Code of Conduct
- Be respectful and inclusive
- Help others learn
- Accept constructive feedback
- Focus on educational value

## Questions or Ideas?
Open an issue to discuss:
- New lab topics
- Structure improvements
- Resource suggestions
- General feedback
